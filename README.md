# AI Salary Prediction System

An end-to-end Machine Learning application that predicts whether a person's annual income exceeds $50,000 based on demographic and employment-related attributes.

## Features

* Income Classification
* LightGBM Machine Learning Model
* Interactive Gradio Dashboard
* Confidence Visualization
* Feature Importance Analysis
* Real-Time Prediction

## Machine Learning Workflow

* Data Exploration
* Data Cleaning
* Feature Engineering
* Model Comparison
* Cross Validation
* Hyperparameter Tuning
* Ensemble Learning
* Model Deployment

## Algorithms Evaluated

* Random Forest
* Extra Trees
* XGBoost
* LightGBM

## Final Selected Model

LightGBM Classifier

### Performance

* Accuracy: 86.9%
* ROC-AUC: 0.9275
* Precision: 0.78
* Recall: 0.64

## Tech Stack

* Python
* Scikit-Learn
* LightGBM
* Gradio
* Pandas
* Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
